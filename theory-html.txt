What does HTML stand for?
    HTML stands for Hyper Text Markup Language.
    With HTML we can describe how our web page will look like.
    We can define different heading, paragraphs, colors, styles, alignment 

What is the structure of an HTML page?
    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>
    <h1>Some heading</h1>
    </body>
    </html>

What is an HTML tag? Examples?
    HTML tag are nothing but elements in brackets like <p> </p>
    Every tag has a start tag and end tag. 
    Only <br> tag has no end tag.
    Examples:
    <p> This is a paragraph </p>
    <html>  html code is enclosed in this tag </html>
    <body> here we can define the web page like color, style </body>

What is an HTML Attribute?
    Attributes saves extra information of the elements
    Like : <a> <img>
    <a href= "www.google.com">Google</a>
    <img src = "abc.png">
    <h1 style="color:blue"> This is a heading </h1>
